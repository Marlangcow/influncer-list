# Inflist (인플리스트)

## 서비스 소개

**Inflist**는 TikTok, YouTube, Instagram, Blog 등 다양한 플랫폼의 인기 인플루언서를 한눈에 비교하고, 데이터 기반으로 빠르게 선정할 수 있는 인플루언서 리스트 서비스입니다.

---

## 핵심 기능

- **플랫폼별 인기 인플루언서 리스트 제공** (TikTok, YouTube, Instagram, Blog)
- **주요 메트릭 표시**: 팔로워/구독자 수, 참여율, 평균 조회수 등
- **카테고리별 분류 및 플랫폼 필터**
- **간편한 검색 기능 및 즐겨찾기**
- **하루 1회 자동 데이터 업데이트**

---

## 주요 타겟 사용자

- 마케팅 에이전시 및 기업 마케터
- 소셜미디어 운영 담당자
- 예비 인플루언서 및 개인 브랜드 시작자

---

## 사용자에게 주는 효과·가치

- 다양한 플랫폼 인플루언서를 한눈에 비교·선정 가능
- 데이터 기반 의사결정으로 캠페인 효율 극대화
- 직관적 UI로 시간 절약 및 빠른 인사이트 확보

---

## 가격 정책

- **Free**: 기본 검색·필터, 월 50회 조회
- **Pro**: 월 29,000원, 무제한 조회, CSV 다운로드, 고급 필터
- **Enterprise**: 연 5,000,000원, API 연동, 전담 지원

---

## 디자인 컨셉

- **톤앤매너**: 모던·클린·데이터 시각화 강조
- **색상**: 배경 화이트 (#FFFFFF), 텍스트 다크 그레이 (#333333), 강조 블루 (#1DA1F2), 그린 (#28A745)
- **레이아웃**: 상단 Hero 섹션 → 플랫폼 탭 UI → 카드형 리스트 그리드 → 주요 기능 강조 섹션 → CTA 버튼

- **인터랙션**: 스크롤 페이드인, 카드 Hover 효과

## 시작하기

1. 저장소 클론 및 의존성 설치
   ```bash
   git clone https://github.com/Marlangcow/influncer-list.git
   cd inflist
   npm install
   ```
2. 개발 서버 실행
   ```bash
   npm run dev
   ```
3. 브라우저에서 [Inflist](https://inflist.vercel.app/) 접속
